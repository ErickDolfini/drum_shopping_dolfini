# Drum Shopping Dolfini

Este é um projeto de e-commerce de instrumentos musicais desenvolvido como parte do meu portfólio acadêmico e profissional. O objetivo é demonstrar habilidades de front-end e organização de projetos web.

## 🚀 Tecnologias Utilizadas
* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: Estilização responsiva e design da interface.
* **JavaScript**: Lógica de interação e dinamismo do site.
* **Git & GitHub**: Controle de versão e hospedagem do código.

## 📌 Sobre o Projeto
O projeto simula uma loja virtual especializada em baterias e pratos, focando em uma interface limpa e intuitiva para o usuário.
